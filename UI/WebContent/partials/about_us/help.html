2.2 Purchase Order

A purchase order serves a simple purpose: It tells some company that you want to purchase some item. purchase orders become permanent records of items that you've ordered. Purchase orders help a purchasing agent to manage incoming orders and pending orders

Purchase Order Screen

Important Data Element
ID Data Elements Details
1 Net Amount The Net Amount shows the total expended for the item.
2 Scheme
3 Void When Void flag ON then Purchase Order soft deleted from record
5 Show Selected (ON/OFF) When Show Selected(ON) then it show only selected product for that Purchase Order
6 Sales Statistics When click on last Sales Statistic dropdown it show the readonly information of Last Sale, Current Sale, Closing Stocks and Pending Orders depend on user selection

How to create new Purchase Order?

1. In Purpleaid Menu, choose Stock Management Purchase Order
The Purchase Order screen appears.

2. Enter the necessary data in the Purchase Order screen areas.

3. Select Product and click on Add Product button, and enter quantity in Quantity input field and Free quantity in Free input field

4. click Ok.

5. If you add more product then follow step 3. and 4.

6. Click on save button to Save the Purchase Order

How to delete PO?

1. Choose Purchase Order

2. Set void flag, Void flag work like a delete but it soft delete.

How to Modify PO?

1. If you wish to change data of purchase order, you must select the desired PO.

2. Change the data

3. Choose Save button to modify the Purchase order

Note

1. If the purchase order still contains incorrect or incomplete items, you can Hold the purchase order and continue processing it later.
